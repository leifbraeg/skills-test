## What this skill does

Takes any text and rewrites it in a specific tone: formal, casual, sarcastic, encouraging, or empathetic.

Great for:
- Converting documentation to friendlier language
- Making code comments more professional
- Rephrasing error messages
- Adapting communication style for different audiences

## How to use

Simply provide text and specify the desired tone. The skill will rewrite it while preserving the core meaning.
